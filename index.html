El Protocolo ArtNet

¿Que en ArtNet?

El protocoloArtNet permite comunicar datos con el protocolo DMX512 y RDM atraves de una red ethernet. Esta basado en una estructura simple de paquetes UDP diseñado para proveer un flujo eficiente de datos.
Aunque este protocolo es de uso libre la empresa dueña de los derechos y el desaroolo es Artistic License Holdings Ltd.
ArtNet fue creado para sobrepasar las limitaciones del protocolo DMX y seguir usando su estructura. Es por ello que permite multiples universos DMX transmitidos en un solo cable Cat5 dentro de una red ethernet.

Modo de Conexionado

  • La configuración del dispositivo, tanto la dirección IP y Subnet, se realiza mediante el webserver integrado, lo que facilita enorme- mente su gestión.
  • Es posible cambiar la descripción del dispositivo para una fácil identificación en la red.
  • Perfecta integraciónen cualquier red informática previamente creada y funcionando.
  • El protocolo es UNICAST, de modo que no consume más recursos de red que los estrictamente propietarios, funcionando perfectamente a través de switches de red, etc.



Mejoras sobre DMX512

  1. Mayor numero de universos con un solo cable
  2. Conexionado por cable Rj45
  3. Configuración sencilla a traves de Switch plug&play
  4. Recepción de errores en la consola de control
